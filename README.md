# PLC Dojo -- Level 2: Applied Logic 

## Overview 
Level 2 moves beyond guided instruction. Each project presents a real-world industrial programming problem that must be solved independently without step-by-step guidance. This is where actual PLC programming competency is built. 

## What Makes Level 2 Different 
In Level 1 the instructor walks through each concept. In Level 2, you receive a problem specification and must write the ladder logic from scratch on your own. All 10 projects in this repository were built independently.

Before this course I have hands-on Allen-Bradley PLC hardware experience including physical PLC swaps, wiring verification, I/O checkout, and server reconnection on automated sorting equipment in a 24/7 industrial environment. This course is building the software programming competency to complement that hardware foundation.
